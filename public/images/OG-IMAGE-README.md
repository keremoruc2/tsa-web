# OG Image

Please add an Open Graph image here named `og-image.jpg` (or `og-image.png`).

**Recommended specifications:**
- Dimensions: 1200 x 630 pixels
- Format: JPG or PNG
- File size: Under 1MB for fast loading
- Content: Include TSA TWENTE logo, name, and a visually appealing background

This image will appear when your website is shared on:
- Facebook
- Twitter/X
- LinkedIn
- WhatsApp
- Slack
- Discord
- And other social platforms

**Tips for a good OG image:**
1. Use the Turkish red (#E30A17) brand color
2. Include the TSA TWENTE logo prominently
3. Add tagline: "Turkish Student Association at University of Twente"
4. Keep text large and readable (it appears as a thumbnail)
5. Use high contrast for visibility
